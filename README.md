前端协议
=======

上下行：

```
+------------------+----------------+---------+
| 后端节点ID (8字节) | 消息长度 (4字节) | 消息内容 |
+------------------+----------------+---------+
```

特殊约定：

1. 客户端发送“消息长度 = 0”，用来显式关闭跟指定后端之间端虚拟连接
2. 客户端和网关之间发送“节点ID = MAX(uint64)”，“消息长度 = 1”，用来进行PING / PONG

后端协议
=======

上下行：

```
+----------------+----------------+---------+
| 消息类型 (1字节) | 消息长度 (4字节) | 消息内容 |
+----------------+----------------+---------+
```

消息类型：

1. 发送给指定客户端

	```
	+--------------------+----------------+---------+
	| 客户端连接ID (8字节) | 消息长度 (4字节) | 消息内容 |
	+--------------------+----------------+---------+
	```

2. 断开指定客户端

	```
	+--------------------+
	| 客户端连接ID (8字节) |
	+--------------------+
	```

3. PING / PONG
